## 구현 목표
### 음성메모/To-do List/ 메모장/ 타이머 기능을 한 앱에 담아 실 서비스 수준의 앱을 개발

## 구현 과정
SwiftUI 구현
∙ AVFoundation(음성)
∙ Local Notification
∙ SwiftUI의 Custom ViewModifier/Animation

## 구현 기능
∙ 음성메모, 알람, Todo 리스트, 메모장 기능

∙ 온보딩화면 토글 구현 및 To do list 완료 목록 삭제

∙ 음성메모 녹음 플레이어 뷰에 애니메이션 추가

∙ 타이머 시간 세팅 커스텀(시/분/초)
